## ğŸ’¡ Key Concepts Explained

### What is Kafka?
Message queue that buffers transactions between producer and consumer. Think of it as a super-fast conveyor belt for data.

### What is XGBoost?
Machine learning algorithm that learns patterns from historical fraud data. Pre-built library - you don't write ML math from scratch.

### What is Feature Engineering?
Extracting meaningful signals from raw data (e.g., "5 transactions in 1 hour" becomes a velocity feature).

### What is Redis?
In-memory cache that stores frequently accessed data for super-fast lookups (sub-millisecond).
